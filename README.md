# Portfólio • Desenvolvedor Fullstack

Site pessoal mostrando projetos, habilidades e informações profissionais.  
Criado para servir como cartão de visita digital — leve, rápido e fácil de atualizar.

---

##  Objetivo

Este portfólio foi desenvolvido para apresentar minhas habilidades e experiências de forma clara e direta.  

---

##  Funcionalidades

- Página inicial com apresentação rápida  
- Seção de projetos  
- Links para redes sociais  
- Layout responsivo (funciona no PC e no celular)  
- Código simples para editar e expandir  
- Estrutura baseada em HTML, CSS e Python

---

##  Tecnologias Utilizadas

Dependendo da sua implementação atual:

- **HTML5**  
- **CSS3**  
- **Python**  
- **Framework Flask**
